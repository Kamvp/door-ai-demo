The Automated Marketing Agency

Turning Door Products into
Million-Dollar Assets

We are building an AI engine that acts as a Senior Architect and Interior Designer. It takes a raw door photo and hallucinates a hyper-realistic, architecturally accurate environment around it.

1. Context Awareness

Automatically detects if the door is Exterior/Interior and adjusts the generated environment (Porch vs Foyer) accordingly.

2. Style Harmony

Never puts a Modern door on a Victorian house. The AI enforces architectural rules to ensure the "Hero Shot" looks authentic.

3. Smart Touch-Up

An advisory engine that helps the user refine details (Wall Color, Handle Style, Glass) without needing to type complex prompts.

1. The "Consultant" JSON Schema

Gemini must return not just options, but the *reasoning* behind them to power the UI advice.

  "refinement_logic": {
    "current_style": "MODERN_MINIMALIST",
    "target_element": "WALL_COLOR",
    "architect_advice": "Since this is a Modern style, avoid warm beiges. Stick to cool neutrals or bold darks for contrast."
  },
  "smart_options": [
    {
      "label": "Concrete Grey",
      "hex": "#9ca3af",
      "reasoning": "Enhances industrial vibe."

      "reasoning": "High contrast with door."
